Campo Tipo (Tipos de tratamentos)
Medicamentos
Tratamento com antibióticos, anti-inflamatórios, antiparasitários, etc.
Cirúrgicos
Procedimentos invasivos, como castrações, remoção de tumores, ortopedia, etc.
Terapias Alternativas
Acupuntura, homeopatia, fisioterapia, entre outros.
Nutricionais
Dietas específicas para condições como obesidade, diabetes ou doenças renais.
Vacinas
Imunizações preventivas.
Controle de Parasitas
Tratamentos contra pulgas, carrapatos, vermes, etc.
Tratamentos Oncológicos
Quimioterapia, radioterapia, cuidados paliativos.
Reabilitação
Fisioterapia ou exercícios específicos após lesões ou cirurgias.
Terapias de Desintoxicação
Tratamento para intoxicações por substâncias químicas ou venenos.
Terapias Paliativas
Tratamentos para melhorar o conforto em condições crônicas ou terminais.
Tratamentos Odontológicos
Limpeza dental, extrações, correções ortodônticas.
Tratamentos Dermatológicos
Uso de cremes, shampoos medicinais ou terapia para condições de pele.
Controle de Zoonoses
Tratamentos para doenças transmissíveis entre humanos e animais.
Campo Descrição (Descrição detalhada do tratamento)
Este campo detalha a abordagem do tratamento. Exemplos:

Medicamentos:
"Antibióticos de largo espectro para infecções bacterianas."
"Anti-inflamatório oral por 7 dias."
Cirúrgicos:
"Cirurgia ortopédica para reparo de fratura femoral."
"Extração de dente comprometido."
Terapias Alternativas:
"Sessões de acupuntura para alívio de dores articulares."
"Homeopatia para controle de ansiedade."
Nutricionais:
"Dieta com ração renal para suporte em insuficiência renal."
"Mudança para dieta hipocalórica visando perda de peso."
Vacinas:
"Aplicação de vacina antirrábica."
"Vacinação anual múltipla."
Reabilitação:
"Exercícios para fortalecimento muscular pós-operatório."
Terapias Paliativas:
"Analgésicos para alívio de dor crônica."
"Hidratação subcutânea para suporte em falência renal."
Campo Duracao_Estimada (Duração típica do tratamento)
Os valores seriam indicativos e variam com o tipo de tratamento. Exemplos:

Curta Duração (1-7 dias):
Administração de medicamentos simples.
Controle de parasitas.
Aplicação de vacinas.
Média Duração (1-4 semanas):
Ciclo de antibióticos ou anti-inflamatórios prolongados.
Fase inicial de reabilitação ou fisioterapia.
Ciclo inicial de quimioterapia.
Longa Duração (1-6 meses ou mais):
Tratamento oncológico completo.
Terapias nutricionais e controle de peso.
Tratamento para condições crônicas (como dermatites ou artrite).

CREATE TABLE Tratamentos (
    ID INT AUTO_INCREMENT PRIMARY KEY, -- Identificador único do tratamento
    Tipo VARCHAR(50) NOT NULL, -- Tipo do tratamento
    Descricao TEXT NOT NULL, -- Descrição detalhada
    Duracao_Estimada VARCHAR(20), -- Duração estimada do tratamento
    Data_Criacao TIMESTAMP DEFAULT CURRENT_TIMESTAMP -- Registro de criação
);
